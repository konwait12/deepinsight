# DeepInsight 项目开发文档

## 1. 项目定位

DeepInsight 是面向深度学习实验流程的可视化分析平台，覆盖数据管理、模型训练、运行日志解析、矩阵化可视化分析、AI 辅助复盘、云端素材管理和社区知识沉淀。

当前产品重点是把“训练数据、模型、分析结果、保存视图、上传文件、AI 对话材料”统一成可追踪、可复用、按用户隔离的研究资产。

## 2. 技术架构

### 前端

- Vue 3 + TypeScript + Vite
- Pinia 管理主题与登录状态
- Vue Router 管理系统总览、数据管理、训练、可视化、AI、云端中心、知识库、论坛、后台等页面
- Element Plus 提供基础交互组件
- ECharts / ECharts GL / AntV G6 支撑可视化图表和知识图谱
- Vue I18n 支持中英文界面

### 后端

- Spring Boot 3.2 + Spring Security + JWT
- Spring Data JPA 管理核心实体
- JdbcTemplate 用于矩阵分析、云端素材、可视化保存等动态表查询
- MySQL 存储用户、训练、数据、可视化结果、云端文件夹与素材
- Redis 存储 AI 记忆、会话缓存等短期上下文

### 安全模型

- 登录认证采用 JWT，无状态访问 API。
- 密码使用 BCrypt 哈希存储。
- `DB_PASSWORD`、`JWT_SECRET` 和模型 API Key 只允许通过环境变量、本地忽略文件或后台配置保存，不进入 Git。
- AI 配置接口仅允许管理员访问，返回时会脱敏 `apiKey`，编辑旧配置时留空可保留原 Key。
- 数据库导出脚本默认清空 `ai_configs.api_key`，只有显式设置导出密钥开关才会包含真实 Key。
- 私有云端素材通过认证接口下载，不暴露静态公开路径。
- 云端数据查询和写入均带 `user_id` 限定，保证用户隔离。
- CORS 默认仅允许本地开发域名，可通过 `APP_CORS_ALLOWED_ORIGINS` 配置生产域名。
- 上传文件限制大小、危险扩展名、MIME 类型和路径穿越。

## 3. 核心模块

### 系统总览

系统总览聚合训练任务、数据资产、云端素材和安全状态。首页提供核心工作区入口、运行中训练任务、数据资产健康度、最近训练结果和云端素材入口。

### 数据管理

数据管理不再只覆盖数据集，而是统一管理：

- 数据集
- 训练任务
- 上传运行与解析日志
- 标量、图片、文本、音频、直方图、嵌入、PR/ROC、超参数、性能记录
- 保存的矩阵分析结果
- 保存的可视化视图
- 已进入云端的素材

支持筛选、搜索、选择同步云端和查看资产可用性。

### 云端中心

云端中心是与知识库同级的完整页面，也是各工作区的小型云端入口来源。

已实现能力：

- 服务器数据库存储云端素材
- 用户级文件夹树
- 新建、重命名、删除、移动文件夹
- 上传任意合规文件到当前文件夹
- 拖动素材或文件夹完成归档
- 素材重命名、删除、下载
- 图片封面和文字封面两种展示模式
- 列表模式与封面墙模式
- 可被 AI 工作室读取为上下文

### 模型训练

训练模块支持选择官方模型或用户模型，配置数据集、学习率、批次、轮次、优化器等参数，创建训练任务并查看实时训练指标。

### 可视化分析

可视化分析支持一对多和多对多矩阵分析。模块覆盖标量、图片、音频、文本、直方图、嵌入、PR/ROC、超参数、模型结构和性能分析。

分析结果可保存到本地或云端，并可导入 AI 对话继续分析。

### AI 工作室

AI 工作室支持多轮对话、工作区资源注入、文件/图片上下文、站内模型/数据/分析结果读取、推理层级选择和历史会话管理。

AI 对话可以读取云端素材、数据资产、训练记录和可视化分析结果，形成新的对话记录。

### 知识库与社区论坛

知识库提供文章阅读和知识图谱展示。社区论坛支持文章浏览、发帖、评论、模式切换和官方内容展示。

## 5. 数据库重点表

- `users`：用户与角色
- `datasets`：数据集
- `training_jobs` / `training_steps`：训练任务与训练过程
- `experiment_runs`：上传运行
- `scalar_logs`、`image_logs`、`audio_logs`、`text_logs`、`histogram_data`、`embedding_data`、`pr_curve_data`、`roc_curve_data`、`hparam_data`、`profiler_data`：可视化日志
- `visual_saved_analysis_records`：矩阵分析保存记录
- `visual_saved_views`：可视化视图保存记录
- `ai_workspace_folders`：云端文件夹
- `ai_workspace_items`：云端素材，包含文本、图片、文件、数据资产快照等
- `ai_configs`：AI 模型服务配置
- `conversations` / `chat_messages`：AI 会话记录
- `forum_posts` / `forum_comments`：论坛内容
- `knowledge_nodes` / `knowledge_docs` / `knowledge_articles`：知识库内容

## 6. 开发与验证

常用命令：

```bash
npm run dev
npm run lint
npm run build
cd backend && ./mvnw.cmd -DskipTests package
```

本地启动前复制 `.env.example` 为 `.env.local`，至少设置 `DB_PASSWORD` 和 32 字节以上的 `JWT_SECRET`。不要提交 `.env.local`、数据库导出、Redis dump、日志、构建产物或浏览器测试 profile。

本次验证状态：

- `npm run lint` 通过
- `npm run build` 通过
- `backend/.mvn/wrapper` 下 Maven Wrapper 打包通过
